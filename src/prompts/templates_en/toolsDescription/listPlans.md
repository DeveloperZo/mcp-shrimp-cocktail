List all plans in a project with their statistics and current status.

Use this tool to:
- View all available plans in a project
- See task statistics for each plan
- Identify the currently active plan
- Compare plan progress and completion rates

Parameters:
- projectName (required): Project to list plans from
- includeStats (optional): Include task statistics (default: true)
- status (optional): Filter by status - "active", "archived", or "all" (default: "all")
- sortBy (optional): Sort by "name", "createdAt", or "updatedAt" (default: "updatedAt")
- sortOrder (optional): Sort order - "asc" or "desc" (default: "desc")
