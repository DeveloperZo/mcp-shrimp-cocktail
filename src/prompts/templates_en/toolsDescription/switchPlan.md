Switch the active plan within a project. All subsequent task operations will work within the selected plan.

Use this tool to:
- Change the working context to a different plan
- Switch between development phases or strategies
- Focus on a specific set of tasks
- Navigate between parallel workflows

Parameters:
- projectName (required): Project containing the plan
- planName (required): Name or ID of the plan to switch to

Example: Switch to the "MVP Development" plan to work on initial features
