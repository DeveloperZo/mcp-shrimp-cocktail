Delete a plan and all its associated tasks. This operation requires confirmation and creates a backup before deletion.

Use this tool to:
- Remove completed or obsolete plans
- Clean up failed experiments or approaches
- Reorganize project structure
- Archive old planning cycles

Parameters:
- projectName (required): Project containing the plan
- planName (required): Name or ID of the plan to delete
- confirm (required): Must be true to confirm deletion

Example: Delete an abandoned feature plan after pivoting strategy
