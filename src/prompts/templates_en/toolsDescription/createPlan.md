Create a new plan within a project. Plans organize tasks into logical groups or phases within a project.

Use this tool to:
- Create execution plans for different scenarios or strategies
- Organize tasks by phase, sprint, or release
- Maintain multiple parallel workflows in one project
- Copy existing plan structure for similar workflows

Parameters:
- projectName (required): Project where the plan will be created
- name (required): Name for the new plan
- description (optional): Detailed description of the plan's purpose
- tags (optional): Tags for categorizing the plan
- copyFromPlan (optional): ID of existing plan to copy tasks from

Example: Create a development plan for a new feature
