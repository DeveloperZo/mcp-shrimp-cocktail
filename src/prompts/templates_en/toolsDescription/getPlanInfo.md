Get detailed information about a specific plan including statistics, metadata, and current status.

Use this tool to:
- View comprehensive plan details and metrics
- Check task completion progress
- Review plan description and tags
- Understand plan context and purpose

Parameters:
- projectName (required): Project containing the plan
- planName (optional): Name or ID of the plan (defaults to current plan if not specified)

Example: Get details about the current development plan including task statistics
