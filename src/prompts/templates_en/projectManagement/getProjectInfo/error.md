## Common Issues

- **Data access problems** - Issues reading project files
- **Permission errors** - Insufficient access to project directory
- **File corruption** - Project metadata may be damaged
- **System resource issues** - Memory or disk space problems

## Troubleshooting Steps

1. **Verify project exists:** `listProjects()` to check all projects
2. **Check file permissions** on project directory
3. **Try other projects** to test if it's a specific project issue
4. **Restart application** to reset file handles
5. **Check system resources** (disk space, memory)

## Alternative Actions

- **List all projects:** `listProjects()` to see system status
- **Switch to different project:** `switchProject("other-project")`
- **Create new project:** `createProject("Backup Project")` if needed

## Data Recovery

- **Check backup files** in the memory directory

- **Avoid making changes** until the issue is resolved
