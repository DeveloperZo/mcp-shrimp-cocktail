# 🔒 項目無法刪除

**項目：** {{projectName}}

## 保護通知

此項目受系統保護政策限制，無法刪除。

## 限制存在的原因

- **活躍內容** - 項目包含任務、計劃或重要資料
- **系統依賴** - 其他組件可能引用此項目
- **資料安全** - 防止意外丟失關鍵項目資料
- **管理政策** - 項目被系統配置標記為受保護

## 替代行動

如果您需要清理或重新組織此項目：

- **清除任務：** 使用任務管理工具移除個別任務
- **歸檔內容：** 首先將重要任務移動到其他項目
- **創建新項目：** 為未來工作創建新項目
- **重置內容：** 移除任務和計劃但保留項目結構
- **聯繫管理員：** 如絕對必要，請求項目刪除

## 創建替代項目

```
createProject("我的新項目", "項目用途描述")
```

這將創建一個您可以用於工作的新項目，同時保留受保護項目的資料。
