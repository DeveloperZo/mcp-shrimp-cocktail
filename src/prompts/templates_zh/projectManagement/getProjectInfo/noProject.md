# ❌ 需要項目名稱

## 問題

您的請求中未指定項目名稱。

## 這意味著什麼

- **項目名稱缺失** - 所有操作都需要明確指定項目
- **請求格式無效** - 項目參數是必需的
- **系統要求明確命名** - 所有項目操作都需要

## 如何解決

1. **列出所有項目** 查看可用項目：
   ```
   listProjects()
   ```

2. **按名稱獲取特定項目資訊：**
   ```
   getProjectInfo("項目名稱")
   ```

3. **如有需要創建新項目：**
   ```
   createProject("我的項目")
   ```

4. **在所有操作中使用明確的項目名稱：**
   ```
   # 始終指定項目名稱
   getProjectInfo("我的項目")
   listTasks("我的項目")
   addTask("我的項目", "任務描述")
   ```

## 系統設計

此系統要求所有操作明確指定項目，以確保：

- **明確意圖** - 您始終知道正在使用哪個項目
- **資料安全** - 不會在錯誤的項目上進行意外操作
- **多項目支持** - 同時在多個項目中工作
- **一致行為** - 所有工具的工作方式相同

請指定項目名稱並重新嘗試您的請求。
