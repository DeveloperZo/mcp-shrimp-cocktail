永久刪除項目及其所有相關數據。

**⚠️ 警告：此操作不可逆轉，將永久刪除所有項目數據，包括任務、配置和元數據。**

此工具提供安全的項目刪除，具有全面的安全檢查和確認要求。

**用法：**
- 永久刪除項目及其所有相關數據
- 為了安全需要明確確認
- 刪除前創建自動備份
- 處理項目依賴關係和引用
- 如果刪除當前項目則更新系統上下文

**參數：**
- `projectName`（必需）：要刪除的項目名稱
- `confirmDelete`（必需）：必須為 true 以確認刪除
- `backup`（可選）：刪除前創建備份（默認：true）

**返回值：**
- 刪除確認及備份信息
- 已刪除數據摘要（任務計數、文件計數）
- 如果刪除了當前項目的上下文更新信息
- 恢復信息和備份位置

**安全功能：**
- 需要明確的確認標誌
- 刪除前自動創建備份
- 在嘗試刪除前驗證項目是否存在
- 檢查活動任務並提供警告
- 防止刪除系統默認項目

**被刪除的數據：**
- 項目內的所有任務
- 項目配置和元數據
- 項目特定文件和目錄
- 任務歷史和完成記錄

**示例：**
```
deleteProject("舊項目", {
  confirmDelete: true,
  backup: true
})
```
