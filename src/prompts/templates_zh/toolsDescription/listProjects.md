列出所有可用項目及其元數據和狀態信息。

此工具提供系統中所有項目的全面概覽，包括項目狀態、任務計數和活動信息。

**用法：**
- 列出所有項目及其詳細元數據
- 顯示當前活動項目及指示器
- 顯示項目統計和任務計數
- 提供項目創建日期和最後活動時間
- 支持過濾和排序選項

**參數：**
- `includeStats`（可選）：包括任務計數統計（默認：true）
- `sortBy`（可選）：排序順序 - "name"、"created"、"modified"（默認："name"）
- `filterTags`（可選）：按特定標籤過濾項目

**返回值：**
- 所有項目的完整列表及元數據
- 當前項目指示器和狀態
- 項目統計，包括任務計數
- 項目管理的快速操作建議

**項目信息包括：**
- 項目 ID 和名稱
- 描述和標籤
- 創建和修改日期
- 任務統計（總計、已完成、待處理）
- 項目狀態（活動、已歸檔等）

**示例：**
```
listProjects({
  includeStats: true,
  sortBy: "modified"
})
```
