將當前活動項目上下文切換到不同的項目。

此工具更改活動項目上下文，影響所有後續任務操作。所有任務管理命令將在新選定的項目範圍內運行。

**用法：**
- 將當前活動項目更改為指定項目
- 更新所有任務操作的系統上下文
- 提供項目切換確認
- 顯示上下文更改對操作的影響

**參數：**
- `projectName`（必需）：要切換到的項目名稱
- `confirm`（可選）：安全確認標誌（默認：false）

**返回值：**
- 項目切換確認，包含之前和新項目名稱
- 更新的項目上下文信息
- 上下文更改的影響摘要
- 新項目上下文中的可用操作

**上下文影響：**
- 所有後續任務操作使用新項目
- 任務列表僅顯示新項目的任務
- 任務創建將任務添加到新項目
- 應用項目特定配置

**安全功能：**
- 在切換前驗證目標項目是否存在
- 提供上下文更改的清晰確認
- 顯示切換前後的項目詳情
- 保留先前項目信息供參考

**示例：**
```
switchProject("網頁應用開發", {
  confirm: true
})
```
