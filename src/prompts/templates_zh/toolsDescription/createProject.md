創建具有指定名稱和配置選項的新項目。

此工具在多項目架構中啟用項目創建，允許用戶將任務和工作組織到獨立的項目上下文中。

**用法：**
- 創建具有唯一名稱和元數據的新項目
- 支持可選的描述、標籤和配置
- 可以從現有項目複製結構
- 自動處理項目目錄創建
- 提供設置為當前活動項目的選項

**參數：**
- `projectName`（必需）：新項目的唯一名稱
- `description`（可選）：關於項目目的的描述性文本
- `tags`（可選）：項目組織的標籤數組
- `copyFromProject`（可選）：從現有項目複製結構
- `setCurrent`（可選）：創建後將此項目設為活動項目

**返回值：**
- 項目創建確認和項目 ID
- 項目元數據和配置詳細信息
- 如果 setCurrent 為 true，則更新當前項目狀態

**示例：**
```
createProject("網頁應用開發", {
  description: "使用 React 的前端網頁應用",
  tags: ["web", "react", "frontend"],
  setCurrent: true
})
```
